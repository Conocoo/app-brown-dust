# 스킬 목록

이 문서는 게임에 구현된 모든 스킬의 상세 정보를 정리합니다.

> 각 스킬은 **템플릿**(구조 + 기본값)으로 정의되며, 용병 데이터에서 `effects` 오버라이드로 수치를 덮어쓸 수 있습니다.
> 아래 표의 수치는 템플릿 기본값이며, 실제 적용값은 용병 데이터(`src/data/mercenaries/`)를 참고하세요.

## 정정당당 (`fair_and_square`)

| 항목 | 값 |
|------|------|
| 타이밍 | after_attack (일반 공격 후) |
| 대상 | 다음 아군 (next_ally) |
| 지속 | 16턴 |
| 효과 1 | 임시생명력 — 시전자 maxHP × 지원력 × 100% (지원력 스케일링, 능력치 강화) |
| 효과 2 | 공격력 증가 — 지원력 × 85% (지원력 스케일링, 능력치 강화) |
| 비고 | 임시생명력 제거 시 연동된 공격력 증가도 함께 제거됨 (linkedBuffId) |
