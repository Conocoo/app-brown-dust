# 스킬 목록

이 문서는 게임에 구현된 모든 스킬의 상세 정보를 정리합니다.

---

## 상급 화상 (`advanced_burn`)

| 항목 | 값 |
|------|------|
| 타이밍 | after_attack (일반 공격 후) |
| 대상 | 적 |
| 지속 | 5턴 |
| 효과 | 매 턴 시전자 ATK × 25% 지속피해 + 받는 피해량 +20% |
| 분류 | dot (지속 피해 면역으로 차단됨) |
| 중첩 | 가능 |
| 비고 | 1개의 상태효과로 표시됨 (DoT + 받피증이 하나로 묶임) |

---

## 상급 극강의 일격 (`advanced_fatal_strike`)

| 항목 | 값 |
|------|------|
| 타이밍 | after_attack (일반 공격 후) |
| 대상 | 적 |
| 효과 1 | 공격력 × 50% 추가피해 (즉시) |
| 효과 2 | 해당 턴 내 적 사망 시 자신의 최대HP × 50% 즉시회복 |
| 비고 | "해당 턴"은 일반공격 + 모든 스킬 대미지 포함. 범위공격 시 회복은 1회만 발동 |

---

## 보호막 (`shield`)

| 항목 | 값 |
|------|------|
| 타이밍 | after_attack (일반 공격 후) |
| 대상 | 자신 |
| 지속 | 12턴 |
| 효과 | 받는 피해량 -25% |
| 분류 | shield (무효화로 제거 가능) |
