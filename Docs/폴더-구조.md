## 핵심 파일 요약

| 파일 | 한 줄 설명 |
|------|-----------|
| `src/types/game.ts` | BattleCharacter, StatusEffect, AttackTargetType, AttackRange, BuffType, DebuffClass, 전투 로그, 게임 상태 |
| `src/types/skill.ts` | 스킬 정의 (Skill, SkillEffect + duration/buffType/debuffClass/ignoreImmunity/triggerSkill) |
| `src/types/mercenary.ts` | 용병 템플릿 (MercenaryTemplate + agility, attackTarget, attackRange, rangeSize) |
| `src/data/skills/index.ts` | 스킬 레지스트리 (Map 기반 ID 조회) |
| `src/data/mercenaries/index.ts` | 용병 레지스트리 (getMercenaryById, getAllMercenaries, getMercenariesByStar) |
| `src/data/runeValues.ts` | 룬 수치 상수 (메인/부가 옵션 기준값, 이 파일만 수정하면 룬 밸런스 조정 가능) |
| `src/logic/battle.ts` | 전투 루프 (라운드 진행, passive 스킬 발동, statusEffects 초기화) |
| `src/logic/targeting.ts` | 타겟 선택 (행 기반 순환 탐색, 범위 대상 목록 계산) |
| `src/logic/damage.ts` | 데미지 계산 (버프/디버프 반영 스탯, 보호막 곱연산, 받는 피해량 증가) |
| `src/logic/turn.ts` | 턴 실행 (CC/면역 판정, 상태 효과, 무효화/정화, 반사반격, 이로운→해로운 순서) |
| `src/logic/rune.ts` | 룬 스탯 적용 (BattleCharacter 생성 시 룬 옵션을 기본 스탯에 반영) |
| `src/components/Board.tsx` | 3x12 격자 보드 (플레이어 6칸 + 간격 + 적 6칸) |
| `src/components/Cell.tsx` | 격자 한 칸 (이모지, HP 바, 캐스팅/쓰러짐 표시) |
| `src/components/CharacterInfo.tsx` | 캐릭터 정보 패널 (3단 레이아웃: 초상화, 스탯/룬, 스킬 슬롯 + 툴팁) |
| `src/components/CharacterList.tsx` | 배치 단계에서 용병 선택 카드 목록 |
| `src/components/OrderSetter.tsx` | 공격 순서 지정 UI |
| `src/components/BattleLog.tsx` | 전투 로그 (최신 상단, 팀 구분, 라운드 구분선) |
| `src/App.tsx` | 메인 파일 (게임 흐름, 우클릭/드래그, 전투 속도 제어) |
