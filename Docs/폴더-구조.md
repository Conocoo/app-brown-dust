# 폴더 구조

```
app-brown-dust/
├── Docs/                          # 프로젝트 설명서 (지금 보고 있는 폴더)
├── Docs_TODO/                     # 설계 문서 (전투 시스템 설계서 등)
├── node_modules/                  # npm이 설치한 라이브러리들 (직접 수정 X)
├── public/                        # 정적 파일 (빌드 시 그대로 복사됨)
│   └── images/                    # 이미지 리소스 (용병 일러스트 등)
├── dist/                          # 빌드 결과물 (npm run build 후 생성)
├── src/                           # 소스 코드 (실제 개발하는 곳)
│   ├── types/
│   │   ├── game.ts                # 게임 타입 (BattleCharacter, 전투 로그, 게임 상태)
│   │   ├── skill.ts               # 스킬 타입 (Skill, SkillEffect, SkillTiming)
│   │   └── mercenary.ts           # 용병 타입 (MercenaryTemplate, StarRating)
│   ├── data/
│   │   ├── skills/                # 스킬 데이터
│   │   │   ├── index.ts           # 스킬 레지스트리 (getSkillById, getAllSkills)
│   │   │   ├── magic-bolt.ts      # 마법 화살 (mage용, before_attack)
│   │   │   └── bless.ts           # 축복 (support용, before_attack)
│   │   └── mercenaries/           # 용병 데이터 (성급별 폴더)
│   │       ├── index.ts           # 용병 레지스트리 (getMercenaryById, getAllMercenaries)
│   │       ├── 3star/             # 3성 용병
│   │       │   ├── warrior.ts     # 검투사
│   │       │   ├── archer.ts      # 궁수
│   │       │   ├── healer.ts      # 성직자
│   │       │   └── tank.ts        # 수호자
│   │       └── 5star/             # 5성 용병
│   │           └── mage.ts        # 마법사
│   ├── logic/
│   │   ├── battle.ts              # 전투 루프 (simulateBattle)
│   │   ├── targeting.ts           # 타겟 선택 (findTarget, findNextAlly)
│   │   ├── damage.ts              # 데미지 계산 (calculateDamage)
│   │   └── turn.ts                # 턴 실행 (스킬 기반 executeTurn)
│   ├── components/
│   │   ├── Board.tsx              # 3x12 격자 보드 (양 팀 사이 간격 포함)
│   │   ├── Cell.tsx               # 보드의 한 칸 (HP 바, 쓰러짐 X, 우클릭/드래그 지원)
│   │   ├── CharacterList.tsx      # 용병 선택 목록 (카드 형태, 드래그 배치)
│   │   ├── OrderSetter.tsx        # 공격 순서 지정 UI (좌클릭 부여, 우클릭 해제)
│   │   └── BattleLog.tsx          # 전투 로그 (최신 상단, 팀 구분, 속도 제어)
│   ├── App.tsx                    # 메인 게임 화면 (상태 관리, 게임 흐름)
│   ├── App.css                    # 게임 화면 스타일
│   ├── index.css                  # 기본 스타일 (배경색, 폰트)
│   ├── main.tsx                   # 앱 시작점 (React를 HTML에 연결)
│   └── vite-env.d.ts              # Vite 타입 선언 (자동 생성, 수정 X)
├── index.html                     # HTML 페이지 (React 앱이 여기에 렌더링됨)
├── package.json                   # 프로젝트 설정 (이름, 라이브러리 목록, 실행 명령어)
├── tsconfig.json                  # TypeScript 설정
├── vite.config.ts                 # Vite 설정
└── CLAUDE.md                      # AI 어시스턴트용 지시 파일 (게임과 무관)
```

## 핵심 파일 요약

| 파일 | 한 줄 설명 |
|------|-----------|
| `src/types/game.ts` | BattleCharacter, 전투 로그, 게임 상태 등 핵심 타입 |
| `src/types/skill.ts` | 스킬 정의 (Skill, SkillEffect, SkillTiming, SkillTargetType) |
| `src/types/mercenary.ts` | 용병 템플릿 (MercenaryTemplate, StarRating) |
| `src/data/skills/index.ts` | 스킬 레지스트리 (Map 기반 ID 조회) |
| `src/data/mercenaries/index.ts` | 용병 레지스트리 (성급별 폴더에서 등록) |
| `src/logic/battle.ts` | 전투 루프 (라운드 진행, passive 스킬 발동) |
| `src/logic/targeting.ts` | 타겟 선택 (행 기반 순환 탐색) |
| `src/logic/damage.ts` | 데미지 계산 (`max(1, atk - def)`) |
| `src/logic/turn.ts` | 턴 실행 (before_attack → 일반공격 → after_attack) |
| `src/components/Board.tsx` | 3x12 격자 보드 (플레이어 6칸 + 간격 + 적 6칸) |
| `src/components/Cell.tsx` | 격자 한 칸 (이모지, HP 바, 캐스팅/쓰러짐 표시) |
| `src/components/CharacterList.tsx` | 배치 단계에서 용병 선택 카드 목록 |
| `src/components/OrderSetter.tsx` | 공격 순서 지정 UI |
| `src/components/BattleLog.tsx` | 전투 로그 (최신 상단, 팀 구분, 라운드 구분선) |
| `src/App.tsx` | 메인 파일 (게임 흐름, 우클릭/드래그, 전투 속도 제어) |
