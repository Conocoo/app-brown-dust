# 프로젝트 개요

## 이 프로젝트는 뭔가요?

**브라운더스트 전투 시뮬레이터**는 웹 브라우저에서 플레이하는 간단한 전투 게임입니다.

- 격자판(3x12) 위에 용병을 배치하고
- 공격 순서를 직접 지정한 뒤
- 자동으로 전투가 진행되며 전투 로그와 함께 승/패 결과를 확인할 수 있습니다

## 사용된 기술

| 기술 | 역할 | 쉽게 말하면 |
|------|------|-------------|
| **Node.js** | JavaScript 실행 환경 | 컴퓨터에서 JavaScript를 돌릴 수 있게 해주는 프로그램 |
| **npm** | 패키지 관리자 | 다른 사람이 만든 코드(라이브러리)를 쉽게 설치해주는 도구 |
| **Vite** | 개발 서버 & 빌드 도구 | 코드를 수정하면 브라우저에 바로 반영해주는 도구 |
| **React** | UI 라이브러리 | 화면(버튼, 격자판 등)을 만드는 데 사용 |
| **TypeScript** | JavaScript + 타입 | JavaScript에 "이 변수는 숫자야" 같은 타입 정보를 추가한 언어 |
| **CSS** | 스타일 | 색상, 크기, 레이아웃 등 화면의 디자인을 담당 |
