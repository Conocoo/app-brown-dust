# 실행 방법

## 사전 준비

Node.js가 설치되어 있어야 합니다. 설치 확인:

```bash
node --version
npm --version
```

버전 번호가 출력되면 정상입니다. 설치가 안 되어 있다면 https://nodejs.org 에서 LTS 버전을 다운로드하세요.

## 1. 라이브러리 설치

프로젝트 폴더에서 터미널을 열고 아래 명령어를 실행합니다. (최초 1회만 하면 됩니다)

```bash
npm install
```

## 2. 개발 서버 실행

```bash
npm run dev
```

실행하면 아래와 같은 메시지가 나옵니다:

```
VITE v6.x.x  ready in xxx ms
➜  Local:   http://localhost:5173/
```

## 3. 브라우저에서 접속

웹 브라우저(Chrome, Edge 등)를 열고 주소창에 입력:

```
http://localhost:5173
```

게임 화면이 나타나면 성공입니다!

## 4. 서버 종료

터미널에서 `Ctrl + C`를 누르면 개발 서버가 종료됩니다.

## 5. 빌드 (배포용)

나중에 다른 사람에게 공유하고 싶을 때 사용합니다:

```bash
npm run build
```

`dist/` 폴더에 완성된 파일이 생성됩니다.
