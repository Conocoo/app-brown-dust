# 게임 사용법

## 게임 흐름

게임은 4단계로 진행됩니다:

```
1. 배치  →  2. 순서 지정  →  3. 전투  →  4. 결과 확인
```

### 1단계: 배치

1. 하단 **용병 목록**에서 용병 카드를 클릭해서 선택합니다
2. 보드 **왼쪽 파란색 영역**(내 팀, 6칸)의 빈 칸을 클릭하면 해당 위치에 배치됩니다
3. 용병 카드를 **드래그**해서 빈 칸에 놓아도 배치됩니다
4. 이미 배치된 캐릭터를 **드래그**해서 다른 빈 칸으로 이동할 수 있습니다
5. 배치된 캐릭터를 **우클릭**하면 **배치가 취소**됩니다
6. 상대 팀(오른쪽 빨간색 영역)은 **랜덤으로 3명**이 자동 배치됩니다
7. **"순서 지정으로"** 버튼을 클릭하면 다음 단계로 이동합니다

### 2단계: 순서 지정

- **좌클릭**: 순서 부여 (1, 2, 3...)
- **우클릭**: 순서 해제 (해당 캐릭터 이후 순서도 모두 취소)
- 모든 용병에 순서를 지정하면 **"전투 시작!"** 버튼이 나타납니다

### 3단계: 전투

- 전투 로그가 순차적으로 표시됩니다 (최신 로그가 상단)
- 보드 상단에 **현재 라운드**가 표시됩니다
- 보드 위 캐릭터의 HP 바가 실시간으로 변합니다
- 쓰러진 용병은 **X** 표시로 표시됩니다
- 로그에서 **[아군]** / **[적군]** 라벨로 팀을 구분합니다

#### 전투 속도 제어

| 버튼 | 기능 |
|------|------|
| **1x / 2x / 4x** | 재생 속도 변경 |
| **일시정지 / 재개** | 전투 일시정지 및 재개 |
| **수동 / 자동** | 수동 모드에서는 "다음" 버튼으로 한 턴씩 진행 |
| **라운드 건너뛰기** | 다음 라운드까지 건너뛰기 |
| **전체 건너뛰기** | 전투를 즉시 완료 |

### 4단계: 결과

- 한쪽 팀이 전멸하면 전투가 끝납니다
- 승리 / 패배 / 무승부 결과가 표시됩니다
- 전투 로그를 확인할 수 있습니다
- **"다시 하기"** 버튼으로 처음부터 다시 시작할 수 있습니다

| **순서 지정** | 순서 부여 | 순서 해제 | - |
| **전투 중** | - | - | - |

## 용병 유형

모든 용병은 4가지 유형 중 하나에 속합니다:

| 유형 | 행동 | 설명 |
|------|------|------|
| **공격형** | 적 공격 | 일반적인 물리 공격을 수행 |
| **방어형** | 적 공격 | 높은 HP와 방어력으로 앞에서 버팀 |
| **지원형** | 아군 지원 | 공격 불가. 다음 순서 아군에게 버프 (추후 구현) |
| **마법형** | 캐스팅 → 공격 | 첫 턴에 캐스팅, 다음 턴에 공격. 캐스팅은 라운드 넘어서도 유지 |

## 용병 스탯표

| 용병 | 아이콘 | 유형 | HP | 공격력(ATK) | 방어력(DEF) |
|------|--------|------|-----|------------|------------|
| 검투사 | ⚔️ | 공격형 | 120 | 30 | 10 |
| 마법사 | 🔮 | 마법형 | 80 | 45 | 5 |
| 궁수 | 🏹 | 공격형 | 90 | 35 | 8 |
| 성직자 | 💚 | 지원형 | 100 | 0 | 12 |
| 수호자 | 🛡️ | 방어형 | 180 | 15 | 25 |

## 전투 규칙

### 데미지 계산
```
데미지 = max(1, 공격자의 ATK - 방어자의 DEF)
```
- 최소 데미지는 항상 **1** (방어력이 아무리 높아도 1은 들어감)

### 턴 순서
1. **상대 팀 1번** → **내 팀 1번** → **상대 팀 2번** → **내 팀 2번** → ... 번갈아 진행
2. 한쪽 인원이 더 많으면 남은 용병이 연속으로 행동합니다
3. 모든 용병이 행동하면 **1라운드가 종료**됩니다
4. 라운드 제한은 없으며, 100라운드 초과 시 무승부 처리됩니다

### 타겟 선택 (행 기반)
1. **같은 행**의 가장 앞에 있는 적을 공격합니다
2. 같은 행에 적이 없으면 **아래 행**의 앞 적을 공격합니다
3. 아래 행에도 없으면 **위 행**의 앞 적을 공격합니다

### 유형별 행동
- **공격형/방어형**: 타겟을 공격합니다
- **지원형**: 공격하지 않고, 자신의 다음 순서 아군에게 지원합니다 (버프 내용은 추후 결정)
- **마법형**: 캐스팅이 안 되어 있으면 캐스팅만 하고 턴 종료. 캐스팅이 되어 있으면 공격 후 캐스팅이 해제됩니다

### 승패 조건
- 상대 팀이 전멸하면 **승리**
- 내 팀이 전멸하면 **패배**
- 100라운드 초과 시 **무승부**
