## 새 환경 셋업 절차

```bash
# 1. 클론
git clone https://github.com/Conocoo/app-brown-dust.git
cd app-brown-dust

# 2. 작업 브랜치 체크아웃
git checkout feature/wiggle-mercenary

# 3. 의존성 설치
npm install

# 4. 개발 서버 실행
npm run dev

# 5. 타입 체크 확인
npm run validate
```

## 프로젝트 주요 커맨드
| 커맨드 | 용도 |
|---|---|
| `npm run dev` | 개발 서버 |
| `npm run build` | 타입 체크 + 빌드 |
| `npm run validate` | 타입 체크만 (빠른 검증) |
| `npm run create-skill -- --config <file>` | 스킬 자동 생성 |
| `npm run create-mercenary -- --config <file>` | 용병 자동 생성 |
