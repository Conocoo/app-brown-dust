# 브라운더스트 전투 시뮬레이터

웹 브라우저에서 플레이하는 브라운더스트 전투 시뮬레이터입니다.

3×12 격자판에 용병을 배치하고, 공격 순서를 지정한 뒤, 자동 전투를 관전합니다.

## 기술 스택

| 기술 | 버전 |
|------|------|
| React | 19 |
| TypeScript | 5.7 |
| Vite | 6 |

상태관리는 React useState만 사용하며, 외부 라이브러리 의존성이 없습니다.

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

개발 서버 실행 후 `http://localhost:5173` 에서 접속합니다.

## 게임 흐름

```
0. 홈 화면  →  1. 배치  →  2. 순서 지정  →  3. 전투  →  4. 결과 확인
```

- **홈 화면**: 전투 시작 및 도감(용병 도감, 스킬 도감) 접근
- **배치**: 용병 목록에서 선택 후 좌측 영역에 클릭/드래그로 배치. 우클릭으로 취소
- **순서 지정**: 좌클릭으로 공격 순서 부여, 우클릭으로 해제
- **전투**: 1x/2x/4x 배속, 일시정지, 수동 모드, 라운드/전체 건너뛰기 지원
- **결과**: 승리/패배/무승부 확인 후 재시작

## 용병 유형

| 유형 | 행동 |
|------|------|
| 공격형 | 적 공격 |
| 방어형 | 높은 HP/DEF로 전열 방어 |
| 지원형 | 스킬로 아군 회복/버프 |
| 마법형 | 적 공격 (캐스팅 미구현) |

## 현재 컨텐츠

- **용병**: 3성 10종 (코델리아, 리즈넷, 아리네스, 칼슨, 마리아, 마야, 위글, 슬론, 제이든, 줄리)
- **스킬**: 41종 (화상, 도발, 무효화, 기절, 저주, 보호막, 중독, 부패, 거인의 일격, 재생, 구속 정화, 반사 신경 등)
- **룬**: 7종 스탯 (HP/ATK/DEF/치명확률/치명피해 등)
- **도감**: 용병 도감, 스킬 도감 (브라운더스트 앱 스타일 UI)

## 프로젝트 구조

```
src/
├── components/    # React UI 컴포넌트 (Board, Cell, BattleLog, MercenaryDex, SkillDex 등)
├── data/
│   ├── mercenaries/  # 용병 데이터 (스탯, 스킬 참조)
│   └── skills/       # 스킬 템플릿 (효과, 타이밍, 대상)
├── logic/         # 전투 로직 (턴 실행, 타겟팅, 데미지 계산)
├── types/         # TypeScript 타입 정의
├── App.tsx        # 메인 앱 (게임 상태 관리)
└── App.css        # 스타일
scripts/           # 자동화 스크립트 (스킬/용병 생성)
Docs/              # 상세 문서 (전투 시스템, 스킬 목록 등)
```

## 커맨드

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 실행 |
| `npm run build` | 타입 체크 + 프로덕션 빌드 |
| `npm run validate` | 타입 체크만 (빠른 검증) |
| `npm run create-skill -- --config <file.json>` | 스킬 자동 생성 |
| `npm run create-mercenary -- --config <file.json>` | 용병 자동 생성 |
