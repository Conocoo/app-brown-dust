## 스킬 목록

### 1. 상급 거인의 일격 (`advanced_giant_strike`)

| 항목 | 값 |
|------|-----|
| timing | `after_attack` (일반 공격 후) |
| target | `enemy_front` |
| duration | 1턴 |
| 설명 | 적에게 추가 피해. 최대생명력 × 40% 추가피해 |
| 하위 스킬 | 회복 (`giant_strike_recovery`) — 적 사망 시 발동 |

**effects:**
- `{ type: 'giant_strike', value: 40, duration: 1 }` — 최대HP 기반 추가 피해
- `triggerSkill: 'giant_strike_recovery'` — 적 사망 시 회복

> ⚠️ 신규 이펙트 타입 필요: `giant_strike` (최대HP % 추가 피해)
> 적 사망 조건부 triggerSkill 로직 필요

### 2. 회복 — 거인의 일격 하위 (`giant_strike_recovery`)

| 항목 | 값 |
|------|-----|
| timing | `after_attack` (일반 공격 후) |
| target | `self` |
| duration | 즉시 |
| 조건 | 추가 피해 유지 시간 동안 적이 사망 시 발동 |
| 설명 | 최대생명력 × 100% 즉시회복 |

**effects:**
- `{ type: 'recovery', value: 100, duration: 0 }` — 최대HP 100% 회복

> ⚠️ 적 사망 조건부 발동 메커니즘 필요 (on_kill 트리거)

### 3. 치명확률 지속 증가 (`crit_rate_sustained_increase`)

| 항목 | 값 |
|------|-----|
| timing | `passive` (전투 시작 전) |
| target | `self` |
| duration | 50턴 |
| 설명 | 치명확률 +14% 초기 부여 + 매 턴 치명확률 +4% 추가 증가 |

**effects:**
- `{ type: 'crit_up', value: 14, duration: 50, buffType: 'stat_enhance' }` — 초기 치명확률 +14%
- `{ type: 'crit_rate_sustained_increase', value: 4, duration: 50, buffType: 'stat_enhance' }` — 매 턴 +4%

> ⚠️ 기존 `crit_damage_sustained_increase` 스킬 참고
> 매 턴 증가하는 누적 버프 로직이 이미 있을 가능성 — 확인 필요

### 4. 재생 (`regeneration`)

| 항목 | 값 |
|------|-----|
| timing | `after_attack` (일반 공격 후) |
| target | `self` |
| duration | 15턴 |
| 설명 | 매 턴 최대생명력 × 14% 회복 |

**effects:**
- `{ type: 'regeneration', value: 14, duration: 15, buffType: 'stat_enhance' }`

> ⚠️ 신규 이펙트 타입 필요: `regeneration` (매 턴 HP% 회복)
> 기존 `recovery`(즉시 회복)와 다르게 지속 회복
