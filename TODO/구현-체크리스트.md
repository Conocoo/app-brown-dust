# 용병 추가 구현 체크리스트

> 대상: 슬론(defender), 제이든(mage), 줄리(support) — 전부 3성

## 1단계: 신규 스킬 파일 생성

### 슬론 스킬
- [ ] `advanced-poison-counter.ts` — 상급 중독 반격 (passive, self)
- [ ] `poison.ts` — 중독 (before_attack, 적 ATK×12%, 공격형 3배, 중첩 가능)
- [ ] `advanced-shield.ts` — 상급 보호막 (after_attack, 받는 피해 -35%, 고통 정화 트리거)
- [ ] `purify-dot.ts` — 고통 정화 (after_attack, DoT 제거, 상급 보호막 하위)
- [ ] `decay.ts` — 부패 (after_attack, 적 최대HP×2.5%, 중첩 가능)
- [ ] `recovery` — 기존 스킬 재사용 검토 (before_attack, 최대HP×10%)

### 제이든 스킬
- [ ] `advanced-giant-strike.ts` — 상급 거인의 일격 (after_attack, 최대HP×40%)
- [ ] `giant-strike-recovery.ts` — 회복 (after_attack, 적 사망 시 최대HP×100%, 하위 스킬)
- [ ] `crit-rate-sustained-increase.ts` — 치명확률 지속 증가 (passive, +14% + 매턴 +4%)
- [ ] `regeneration.ts` — 재생 (after_attack, 매턴 최대HP×14%)

### 줄리 스킬
- [ ] `advanced-cc-purify.ts` — 상급 구속 정화 (after_attack, CC 제거)
- [ ] `advanced-regeneration-grant.ts` — 상급 재생 부여 (after_attack, 매턴 최대HP×10% + 즉시 5%)
- [ ] `reflex-grant.ts` — 반사 신경 부여 (after_attack, 민첩 +50%sp, 약탈/복사 면역)
- [ ] `reflex-def-up.ts` — 방어력 증가 (피격 시, 방어력 +100%, 반사 신경 하위)
