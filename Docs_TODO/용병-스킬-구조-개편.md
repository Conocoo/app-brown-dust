# 용병 스킬 시스템 — 재설계 중

기존 스킬(강타, 축복, 마법 화살)은 모두 삭제됨. 스킬을 처음부터 새로 설계하여 추가 예정.

---

## 현재 상태

- 스킬 데이터 파일: 전부 삭제 (`skills/index.ts`의 레지스트리만 남아있음)
- 모든 용병의 `skillIds`: `[]` (빈 배열)
- 전투 로직(`turn.ts`): 스킬 시스템 코드는 그대로 유지 (데이터만 비어있음)

## 턴 실행 순서

한 용병의 턴은 다음 순서로 진행:

```
1. before_attack 스킬 발동 (이로운 효과 → 해로운 효과 순서)
2. 일반공격 (ATK × (1 - DEF/100))
3. after_attack 스킬 발동 (스킬 번호 순서)
```

- `passive` 스킬은 게임 시작 시 1회 발동 (전투 루프 진입 전)
