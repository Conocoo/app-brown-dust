# 용병 세분화 + 스킬 시스템 구조 개편

구조 개편 완료. 아래는 현재 구현된 스킬 시스템 규칙 정리.

---

## 턴 실행 순서

한 용병의 턴은 다음 순서로 진행:

```
1. before_attack 스킬 발동 (스킬 1~4 중 timing이 before_attack인 것)
2. 일반공격 (공통: max(1, atk - def))
3. after_attack 스킬 발동 (스킬 1~4 중 timing이 after_attack인 것)
```

- `passive` 스킬은 게임 시작 시 1회 발동 (전투 루프 진입 전)
- 스킬 발동 순서: 스킬1 → 스킬2 → 스킬3 → 스킬4 (같은 timing 내)
- 일반공격은 모든 용병 공통 메커니즘 (`max(1, atk - def)`). 별도 데이터 불필요.
